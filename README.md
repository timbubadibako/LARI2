# LARI2: Sync Your Stride. Own the Grid.

LARI2 is a next-generation fitness application that transforms outdoor running into a real-time geospatial territory conquest game. Runners claim physical areas by completing route loops, competing for global dominance.

## 🚀 Core Mechanics
- **Territory Conquest:** Capture real-world locations by running in closed loops.
- **Dynamic Factions:** Join a guild and compete for global leaderboard dominance.
- **Elastic Trails:** High-precision GPS tracking with snap-to-road algorithms.

## 🛠 Tech Stack
- **Frontend:** Flutter & Dart (Riverpod, MapLibre GL).
- **Backend:** Go (Golang) with Echo framework.
- **Database:** PostgreSQL with PostGIS for advanced geospatial intelligence.
